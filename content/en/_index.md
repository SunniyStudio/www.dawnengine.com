## Built for Serious Worlds

DawnEngine is a **commercial real-time 3D engine** built on the open-source **O3DE**, targeting digital twins, simulation, games, and industrial visualization. On top of a mature open foundation we add **double-precision rendering**, **World Partition** large worlds, **HTN explainable AI**, **Cesium real-world geospatial** data, and **domestic OS support** — delivered with professional commercial licensing and support for self-controlled, high-precision large worlds.

## Core Features

DawnEngine delivers a complete capability matrix for large worlds and simulation. See the full [features overview](/en/features/), or jump to the highlights:

- [Built on O3DE](/en/features/built-on-o3de/) — open foundation, self-developed enhancements.
- [Enhanced Double-Precision Rendering](/en/features/double-precision-rendering/) — planet-scale coordinates without jitter.
- [World Partition](/en/features/world-partition/) — seamless worlds with automatic partitioning and streaming.
- [HTN AI System](/en/features/htn-ai/) — explainable agents driven by hierarchical task networks.
- [3D Flight Navigation](/en/features/flight-navigation-3d/) — true 3D aerial pathfinding.
- [Vehicle System](/en/features/vehicle-system/) — physics-driven cars, aircraft, and vessels.
- [Pixel Streaming](/en/features/pixel-streaming/) — cloud rendering, zero-install browser access.
- [Multi-Channel Rendering](/en/features/multi-channel-rendering/) — multi-viewport, multi-display, multi-GPU.
- [Cesium Geospatial](/en/features/cesium-geospatial/) — real Earth, photogrammetry, and 3D Tiles.
- [Domestic OS Support](/en/features/domestic-os/) — Kylin / UOS / HarmonyOS adaptation.
