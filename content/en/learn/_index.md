## Start Learning DawnEngine

Whether this is your first real-time 3D engine or you are migrating from another, this section helps you get productive quickly.

## Getting Started

- **Install & configure**: [request a trial](/en/trial/) to get platform packages and a license key.
- **First project**: create a project, import assets, and run your first scene.
- **Core concepts**: entity-component system, Gem modules, the Atom renderer.

## Advanced Topics

- **Large worlds**: [World Partition](/en/features/world-partition/) and [enhanced double-precision rendering](/en/features/double-precision-rendering/).
- **Geospatial**: data ingestion and display with [Cesium geospatial](/en/features/cesium-geospatial/).
- **AI & vehicles**: [HTN AI system](/en/features/htn-ai/), [3D flight navigation](/en/features/flight-navigation-3d/), and the [vehicle system](/en/features/vehicle-system/).
- **Delivery**: [pixel streaming](/en/features/pixel-streaming/) and [multi-channel rendering](/en/features/multi-channel-rendering/).

## API Reference

Full API docs will be provided alongside releases. In the meantime, see the upstream [O3DE documentation](https://docs.o3de.org/) for foundation APIs.

> Documentation is being expanded continuously. If you get stuck, ask in the [community](/en/community/).
