We are excited to announce that **DawnEngine 1.0** is now available. This is DawnEngine's first stable release, marking the engine's transition to production readiness for large-scale simulation and visualization.

## Highlights

- **Enhanced double-precision rendering**: 64-bit world coordinates keep city- and planet-scale worlds stable and jitter-free far from the origin. See the [feature page](/en/features/double-precision-rendering/).
- **World Partition**: automatic partitioning and view-distance streaming for seamless, collaborative large worlds. See the [feature page](/en/features/world-partition/).
- **HTN AI system**: explainable agent planning based on hierarchical task networks. See the [feature page](/en/features/htn-ai/).
- **Cesium geospatial**: real Earth, photogrammetry, and massive 3D Tiles data.

## Localization

1.0 also ships adaptation support for **Kylin and UOS**, with **HarmonyOS** support entering preview. Learn about [domestic OS support](/en/features/domestic-os/).
