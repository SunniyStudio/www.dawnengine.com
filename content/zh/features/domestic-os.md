## 概述

面向政企与关键行业，**自主可控**是硬性要求。DawnEngine 在国产化方面进行系统适配，支持 **银河麒麟（Kylin）**、**统信 UOS** 与 **鸿蒙 OS** 等国产操作系统，兼容主流国产 CPU 架构，可在信创环境中稳定部署与运行。

## 核心能力

- **国产操作系统**：适配银河麒麟、统信 UOS、鸿蒙 OS。
- **国产处理器**：兼容主流国产 CPU 架构。
- **图形适配**：面向国产 GPU 与 Vulkan 等图形栈进行适配优化。
- **本地化交付**：支持离线/内网部署，满足信创合规与安全要求。
- **成熟底座**：底层基于开源 [O3DE](built-on-o3de/)，降低供应链与底座锁定风险；商业能力与技术支持由 DawnEngine 提供。

## 适用场景

- 政务、国防、能源、交通等对自主可控有要求的关键行业。
- 信创环境下的数字孪生、仿真与可视化平台。
- 需要国产软硬件整体方案的政企客户。

## 协同特性

国产化支持是 DawnEngine 的底层保障，让[Cesium 地理孪生](cesium-geospatial/)、[像素流](pixel-streaming/)等能力可在信创软硬件上落地。了解更多合作与适配进展，请见[生态系统](/ecosystem/)。
