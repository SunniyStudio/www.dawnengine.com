## 开始学习 DawnEngine

无论你是初次接触实时 3D 引擎，还是从其他引擎迁移而来，这里都能帮助你快速上手。

## 入门

- **安装与配置**：通过[申请试用](/trial/)获取对应平台的试用安装包与授权密钥。
- **第一个项目**：创建工程、导入资源、运行第一个场景。
- **基础概念**：组件实体系统、Gem 模块、Atom 渲染器。

## 进阶主题

- **大世界**：[World Partition](/features/world-partition/) 与[增强双精度渲染](/features/double-precision-rendering/)。
- **地理信息**：[Cesium 地理系统](/features/cesium-geospatial/)的数据接入与展示。
- **AI 与载具**：[HTN AI 系统](/features/htn-ai/)、[3D 飞行导航](/features/flight-navigation-3d/)与[载具系统](/features/vehicle-system/)。
- **交付**：[像素流](/features/pixel-streaming/)与[多通道渲染](/features/multi-channel-rendering/)。

## API 参考

完整 API 文档将随版本发布提供。在此之前，可参考上游 [O3DE 文档](https://docs.o3de.org/) 了解底座 API。

> 文档持续完善中。如果你在学习过程中遇到问题，欢迎到[社区](/community/)提问。
